# React Tic Tac Toe
Minimalistic Tic Tac Toe game built with React. AI is driven by a minimax algorithm.

## Usage

* Clone the repository
`https://github.com/LordMathis/react-tic-tac-toe.git`

* Install
`npm install`

* Run development server
`npm run start`

* Build for production
`npm run build`
